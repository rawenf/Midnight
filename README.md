## 🛰️ Core Identity & Features

### 🌌 The Abyss Aesthetic
Choose between **Midnight** (Sleek Dark) and **Abyss** (High-Contrast Onyx) modes. The interface is crafted to disappear, leaving only the archival signals in focus.

### 🔑 Neural Identity (Google Integration)
Zero-friction synchronization. Using **Google Identity Services**, users can establish or reconnect their archival profiles with a single click. Identity handles are unique, verified, and permanent.

### 🧠 AI Visual Intelligence
Powered by **Google Gemini**, every signal transmitted to the void is analyzed in real-time. The neural core suggests categories and identifies visual spectrums (colors) automatically to ensure perfect archival sorting.

### 💬 Private Frequencies (Private Messaging)
End-to-end signal transmission. Send text, archival images, and GIFs through private neural channels. unread indicators and real-time status updates ensure identity continuity.

### 🖼️ High-Fidelity Inspection
Interactive signal exploration with:
- **Neural Zoom:** Scroll or click to inspect high-resolution vectors.
- **Diagnostics:** View spectral data, resolution, and archival metadata for every signal.
- **Reactions:** Like, save, and comment to synchronize with other observers.

## 📥 Archival Configuration

To synchronize this node with your local environment, establish a `.env` file in the root directory:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_FIREBASE_FIRESTORE_DATABASE_ID=(default)

GEMINI_API_KEY=your_google_ai_studio_api_key
```
